Задание. 
Создать проект «База данных заказов». Создать
таблицы «Товары» , «Клиенты» и «Заказы».
Написать код для добавления новых клиентов,
товаров и оформления заказов.

Решение.
Использованы Mysql, JDBC, JSP/JSTL, Servlet, bootstrap.
Реализовано в виде модели интернет-магазина с корзиной товаров.
Корзина хранится в session.

Для доступа в интернет-магазин необходимо авторизоваться.
Есть два интерфейса: пользователь и администратор.
Пользователь: выбирает товары и оформляет заказ.
Администратор: [действия пользователя] + добавляет/удаляет товары, просматривает/удаляет заказы.
  Для действий администратора - вверху ссылка [Панель администратора].

Регистрация новых пользователей не реализована.
